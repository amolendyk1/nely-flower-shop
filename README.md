# The Bud & Bloom — Project Repo

**Client:** Nely

Brief Description 
- This repo contains all pages needed to run the online website, The Bud & Bloom. It includes a home page, a flower library, plant library, bouquet library, a shop where you can purchase a bouquet or plant, a contact page to sign up for bouquet making classes, about me page which talks about the owner, Nely, and a cart which you can purchase the items. 

Link to the live site 
- [repo](https://github.com/amolendyk1/nely-flower-shop) | [live](https://amolendyk1.github.io/nely-flower-shop/) |

What you learned
- From this project, I learned how to make a fully functioning website which includes several pages, links from the home page to those pages, a shopping cart feature, a way to purchase items, and a way to contact our business. I learned how to implement client suggestions ideas into a complete working online business.In addition, I learned about new flowers, plants, and bouquet combinations that I did not have previous knowledge on.
